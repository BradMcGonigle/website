The initial foundation of the personal website establishing the monorepo architecture.

## Highlights

- **Turborepo Monorepo**: Configured pnpm workspaces with Turborepo for efficient builds
- **Next.js 16**: Set up Next.js 16 with App Router and React 19
- **Shared Configurations**: Created reusable configs for TypeScript, ESLint, Prettier, and Tailwind
- **Component Architecture**: Established packages structure for pages, components, and utilities
- **CI/CD Pipeline**: Added GitHub Actions workflow for PR checks including lint, typecheck, and build

## Technical Stack

- Next.js 16 with App Router
- React 19
- TypeScript for type safety
- Tailwind CSS v4 for styling
- pnpm v10 with workspace catalogs
